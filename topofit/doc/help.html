TopoFit Documentation

This addon provides an elegant, non-destructive, and entirely Edit Mode-centric single-mesh workflow for creating corrective shape keys, especially for fitting clean base topologies to raw human scan data.

By operating directly on the main mesh's Edit BMesh shape layers, you avoid complex parenting, armature sync bugs, and visual clutter.

Core Plugin Workflow

The workflow for TopoFit is guided by the main panel, which dynamically displays instructions based on the current state of your mesh. The "Status & Workflow Guidance" section, typically found at the bottom of the TopoFit panel and collapsed by default, provides detailed steps and status updates. Follow these phases to effectively use the addon:

  1. 🧬 Step 1: Designate Landmarks

    Select your base character mesh, enter Edit Mode, and select the key vertices you want to act as landmark control points (e.g., corners of the eyes and mouth, key nose joints, jawline points, joint positions, etc.).

    For symmetrical topologies, you only have to select points on one side of the body. You can use the Select Mirrored operator in the Landmark Setup panel to automatically select equivalent landmarks on the opposite side.

    Once selected, click Add Selected to Landmarks. This registers these vertices into the TopoFit_Landmarks vertex group, which the fitting algorithm uses for guidance.

  2. ⚠️ Step 2: Set Active Shape Key

    To protect your baseline human geometry from destructive edits, the addon prevents landmark adjustments directly on the default "Basis" shape key.

    In the standard Blender Shape Keys panel, select or create a custom target shape key (e.g., "TopoFit_Target") and make sure its value is set to 1.0. The addon also provides a Create & Set Reshape Key operator for convenience.

  3. 🔧 Step 3: Align Landmarks & Symmetrize

    With your landmarks set up and your target shape key active, you are ready to match your character mesh to your scan data:

    • Toggle Landmark Mask: Click this button to temporarily isolate your landmark vertices. The addon applies a non-destructive MASK modifier, hiding all other vertices in both the viewport and Edit Mode so you can work without visual clutter.
    • Reposition: Grab, slide, or snap the visible landmark vertices directly to your target 3D scan surface.
    • Apply Symmetry: If you've positioned landmarks on one side of the body, click this button to instantly mirror those displacements relative to the basis position of their symmetrical partners across the X-axis. This works perfectly even while the mask is active!
    • Revert Selected to Basis: Reset any selected landmarks back to their original position relative to the Basis shape key.
    • Refresh Mirror Map: Click this if you have modified the topology or if symmetrical mirroring is behaving unexpectedly.

  4. 🚀 Step 4: Fit Remaining Mesh

    Once you are satisfied with the landmark placement:

    Click Fit Remaining Mesh (IDW) directly inside Edit Mode. The addon will immediately trigger a geodesic distance-weighted Inverse Distance Weighting (IDW) algorithm. It will calculate the displacement deltas of your landmarks and smoothly interpolate the position of every other non-landmark vertex in your active shape key to perfectly fit the body structure.

    You can adjust the Falloff Power (higher values localize the influence of landmarks; lower values distribute influence more broadly) and the Max Influence Distance in the operator settings box to control the deformation behavior.
